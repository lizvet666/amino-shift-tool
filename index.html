⚙ 設定
店舗ルール（AIプロンプト用）

店舗ごとの詳細ルールを記述します。設定した構造化制約は自動で追記されます。

スタッフルール（AIプロンプト用）

スタッフごとの優先配置・希望などを記述します。

AI エンジン選択

シフト自動生成に使用するAIを選択してください。それぞれのAPIキーを下のパネルで保存してください。

Gemini API キー

Google AI Studio（aistudio.google.com/apikey）で取得したAPIキーを入力してください。
キーはこのブラウザのlocalStorageに保存され、外部に送信されることはありません。

OpenAI API キー（ChatGPT）

OpenAI Platform（platform.openai.com/api-keys）で取得したAPIキーを入力してください。
キーはこのブラウザのlocalStorageに保存され、外部に送信されることはありません。

LINE通知（GAS Webhook URL）

GASのウェブアプリURLを入力すると、シフト確定・公開時にLINEグループへ自動通知されます。
